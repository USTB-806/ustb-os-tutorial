欢迎各位同学选择操作系统课程实践!

在本课程中，你将使用 Rust 语言，从零开始实现一个基于 RISC-V 架构的类 UNIX 操作系统内核. Rust 是一门注重性能与内存安全的现代编程语言，而 RISC-V 则是一种近年来备受关注的开源指令集架构. 通过完成本实验，你将亲手打造一个具备内存安全等多项优势的操作系统内核.
