执行如下命令：

```shell
git clone https://github.com/Source-of-USTB/ustb-os-rustlings.git
```

进入项目目录后，再执行：

```shell
cargo install --force --path .
```

之后你可以执行 `rustlings watch` 命令来启动交互式界面，完成习题.

每完成一道习题，需要删除注释 `// I AM NOT DONE`，否则该题不会被标记为完成.

直至所有练习完成，提交希冀评测.
