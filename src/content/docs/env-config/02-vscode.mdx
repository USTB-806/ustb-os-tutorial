---
title: VSCode 下载与安装
description: 环境配置指导.
---

从 [VSCode 官网](https://code.visualstudio.com/) 下载.

安装 `Docker` 插件，或者 `Dev Containers` + `Container Tools` 插件，由此获取 VSCode 的容器插件支持.

<img src={import.meta.env.BASE_URL + "images/docker.png"} alt="插件安装示例" width="50%" />

然后你需要安装 `rust-analyzer` 插件来增强 Rust 语言的显示. `RISC-V Support`, `Even Better TOML`, `Error Lens` 和 `Git Graph` 插件我们也建议安装.

如果你不想使用 VSCode，也可以使用 RustRover 等其他开发环境.
